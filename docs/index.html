변수와 값

var 키워드로 변수를 선언합니다. 선언과 동시에 값을 할당하거나, 나중에 할당할 수 있습니다.

문법

var <이름>;
var <이름> = <값>;
<이름> = <새 값>;

예시

> var x = 10;
> var name = "Alice";
> var flag = true;
> var empty;
> print empty;
nil
> x = x + 1;
> print x;
11

값의 종류

종류예시출력
정수4242
실수3.143.14
문자열"hello"hello
불리언true / falsetrue / false
없음var x;nil
참고타입을 앞에 쓰는 방식(int x = 1;)은 지원하지 않습니다. var를 사용하세요.
참고상수(const)는 지원하지 않습니다. var로 선언한 변수는 항상 재대입 가능합니다.

조건문

문법

if (<조건>) {
    <참일 때>
} else if (<조건2>) {
    <조건2가 참일 때>
} else {
    <모두 아닐 때>
}

else if와 else는 생략할 수 있으며, else if는 몇 개든 이어붙일 수 있습니다.

주의열린 중괄호 {는 반드시 키워드와 같은 줄에 작성해야 합니다. 새 줄에 {를 쓰면 오류가 발생합니다.
if (x > 0)
    { ... } // 오류  →  if (x > 0) { ... } // 정상
> var score = 85;
> if (score >= 90) {
...     print "A";
... } else if (score >= 80) {
...     print "B";
... } else {
...     print "C";
... }
B
참고elif, elsif는 지원하지 않습니다. else if를 사용하세요.
← 연산자다음: 반복문 →

반복문

문법

for (<초기화>; <조건>; <증감>) {
    <반복할 코드>
}

초기화, 조건, 증감식은 모두 생략할 수 있습니다.

주의열린 중괄호 {는 반드시 for (...)와 같은 줄에 작성해야 합니다. 새 줄에 {를 쓰면 오류가 발생합니다.
for (...)
    { ... } // 오류  →  for (...) { ... } // 정상
> for (var i = 0; i < 5; i = i + 1) {
...     print i;
... }
0
1
2
3
4

while 대체 패턴

> var i = 0;
> for (; i < 3; i = i + 1) {
...     print i;
... }
0
1
2
참고while, do는 지원하지 않습니다. for를 사용하세요.
주의break와 continue는 지원하지 않습니다. 루프 조건을 변수로 제어하세요.
← 조건문다음: 함수 →

함수

문법

func <이름>(<파라미터>, ...) {
    <본문>
    return <값>;
}
주의열린 중괄호 {는 반드시 func 이름(...)과 같은 줄에 작성해야 합니다. 새 줄에 {를 쓰면 오류가 발생합니다.
func add(a, b)
    { ... } // 오류  →  func add(a, b) { ... } // 정상

선언과 호출

> func greet(name) {
...     print "Hello, " + name;
... }
> greet("Alice");
Hello, Alice

반환값

> func add(a, b) {
...     return a + b;
... }
> print add(3, 7);
10

재귀 함수

> func fact(n) {
...     if (n <= 1) return 1;
...     return n * fact(n - 1);
... }
> print fact(5);
120
참고return 없이 끝나거나 return;이면 nil을 반환합니다.
참고함수를 변수에 저장하거나 인자로 전달할 수 없습니다. 이름으로 직접 호출만 가능합니다.
참고Func, function, def, fn 등은 지원하지 않습니다. 반드시 소문자 func를 사용하세요.
← 반복문다음: 배열 →

배열

array(n)으로 크기 n의 배열을 생성합니다. 모든 원소의 초깃값은 nil이며, 인덱스는 0부터 시작합니다.

문법

var <이름> = array(<크기>);
<이름>[<인덱스>] = <값>;  // 쓰기
<이름>[<인덱스>]           // 읽기

생성 · 읽기 · 쓰기

> var arr = array(3);
> arr[0] = 10; arr[1] = 20; arr[2] = 30;
> print arr[0];
10
> print arr;
[10, 20, 30]

배열을 함수에 전달 (참조 전달)

배열은 참조로 전달되므로 함수 안에서 수정하면 밖에서도 반영됩니다.

> func fill(a, val) {
...     a[0] = val; a[1] = val;
... }
> var arr = array(2);
> fill(arr, 7);
> print arr;
[7, 7]
← 함수다음: 스코프와 블록 →

스코프와 블록

중괄호 {}는 새로운 스코프를 만듭니다. 블록 안에서 선언한 변수는 밖에서 보이지 않습니다.

> var x = 1;
> {
...     var x = 2;
...     print x;
... }
2
> print x;
1
참고if, for, func의 중괄호도 동일하게 독립 스코프를 만듭니다.
← 배열다음: 주석 →

주석

// 이후부터 줄 끝까지 무시됩니다.

// 이 줄 전체는 실행되지 않습니다
var x = 10;   // 인라인 주석도 가능
참고/* */ 블록 주석은 지원하지 않습니다.

REPL — 대화형 모드

인자 없이 실행하면 대화형 모드로 진입합니다.

factory

> func double(x) {
...     return x * 2;

REPL 선행 검사

블록 입력 도중 선언된 적 없는 변수를 사용하면 해당 줄을 입력하는 즉시 에러가 표시됩니다. 에러가 난 줄만 스킵되고 나머지 블록은 유지됩니다.

> func add(a, b) {
...     var d = e + 3;
[line 2] Undefined variable 'e'.
...     return a + b;
... }
> print add(3, 4);
7

func fib(n) {

← 런타임 오류다음: 합산 예제 →

1부터 N까지 합산

func sumTo(n) {
    var sum = 0;
    for (var i = 1; i <= n; i = i + 1) {
        sum = sum + i;
    }
    return sum;
}

print sumTo(10);
print sumTo(100);
55 5050

버블 소트

var arr = array(5);
arr[0] = 3; arr[1] = 1; arr[2] = 4; arr[3] = 1; arr[4] = 5;

for (var i = 0; i < 5; i = i + 1) {
    for (var j = 0; j < 4 - i; j = j + 1) {
        if (arr[j] > arr[j + 1]) {
            var tmp = arr[j];
            arr[j] = arr[j + 1];
            arr[j + 1] = tmp;
        }
    }
}
print arr;
[1, 1, 3, 4, 5]

문자열 유틸

func repeat(s, n) {
    var result = "";
    for (var i = 0; i < n; i = i + 1) {
        result = result + s;
    }
    return result;
}

func max(a, b) {
    if (a > b) return a;
    return b;
}

print repeat("ha", 3);
print repeat("-", 10);
